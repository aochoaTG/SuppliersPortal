# Stack Tecnológico - TotalGas

## Backend
- **Framework**: Laravel 12
- **PHP**: 8.2.*
- **Base de datos**: SQL Server 2019
- **ORM**: Eloquent
- **Autenticación**: Laravel Breeze

## Frontend
- **CSS Framework**: Bootstrap 5.3
- **Template**: Zircos Template
- **Icons**: Tabler Icons
- **DataTables**: Yajra Laravel DataTables
- **JavaScript**: jQuery 3.7
- **Alerts**: SweetAlert2 3.x
- **Componentes**: Livewire 3 (opcional)

## Patrones de Desarrollo
- **Arquitectura**: MVC con Service Layer
- **Validación**: Form Requests
- **Rutas**: Resource Controllers
- **Middleware**: Custom middleware para permisos
- **API**: RESTful JSON responses

## Convenciones
- **Idioma**: Español en UI, código en inglés
- **Base de datos**: snake_case
- **Clases**: PascalCase
- **Métodos**: camelCase
- **Vistas**: Blade templates en `resources/views`
- **Horario**: Zona horaria Mexico/Chihuahua
