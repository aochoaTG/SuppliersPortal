# Cuestionario de Validación para Compras y Finanzas

## Propósito
Cerrar la política operativa del presupuesto antes de implementar la unificación técnica. Cada respuesta debe aprobarse como regla oficial del portal.

## Resumen Ejecutivo Propuesto
- Comprometer presupuesto cuando una OC/OCD queda aprobada para ejecución.
- Liberar presupuesto cuando una orden comprometida se rechaza, cancela, devuelve o cierra sin recepción.
- Consumir presupuesto al registrar la remisión o entrega física del proveedor.
- Mantener la facturación fuera del cálculo presupuestal hasta que exista módulo formal.
